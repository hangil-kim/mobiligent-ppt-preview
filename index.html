디자인 구성 요소

제목: 아이콘 원형 배지 + 26px Bold Navy + 우측 그라데이션 밑줄
KPI 카드: 네이비 그라데이션 배경, 원형 아이콘, 골드 대형 숫자, 그림자
정보 카드: 화이트 배경, 좌측 4px 컬러 보더, 아이콘+제목 헤더, 둥근 모서리, 그림자
카드 색상 코드: 블루(기본/기술) | 골드(강조/핵심) | 그린(긍정/성과) | 레드(문제/경고)
다이어그램: 원형 아이콘 배지(상단) → 코어 엔진 바(중앙) → 하단 요소. 화살표로 흐름 표시
프로필: 이니셜 아바타 + 경력 골드 배지 + 좌측 정렬 정보
타임라인: 수평 원형 노드(연도) + 카드 + 매출 강조 숫자. 색상으로 시간 구분
표지/마지막: 네이비 그라데이션 + 반투명 원형 장식 + 골드 디바이더
브랜드 요소: 상단 5px 그라데이션 바 + 좌하단 워터마크 (내용 슬라이드)
